8.JavaScript中的对象：

JavaScript和Java的不同之一在于没有类的概念，而直接是对象的概念。对象定义如下：

var car = {type:”Flat”,model:500};   //声明的过程同时是定义的过程

对象的访问，car.type 或 car[“type”];

对象方法的定义

var car={type:”Flat”,getType:function(){ return this.type;}};

  几个关键点，1.语法上有点类似于，定义了一个变量，但是这个变量是一个函数的感觉。调用是采用 car.getType()；这种函数访问方式 。 2.也有this的关键字。 总体感觉JavaSctip中的对象感觉有点怪。（使用时，貌似可以申明一个初始化的，但是对象貌似只适合单次使用，如何能实现像类那样的多次复用呢？ 推测可能有几个思路，一看JavaScript中有无类似new这样的关键字，二是看是否有复制或克隆的方法，三是通过一个函数，把对象声明封装在内，每次需要使用对象时，用函数来获取一个新的对象，用以取代new关键字.  测试了一下，第三种思路是可行的，但是貌似比较少看见。可以考虑通过js文件来定义一些通用类来做一些事情，但是因为源码是暴露的，这样做可能会存在一些不安全。）
