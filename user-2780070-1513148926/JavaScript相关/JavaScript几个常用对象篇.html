JavaScript几个常用对象篇

javascript对象

1.javascript中所有事物都是对象，另外javascript允许自定义对象。

2.对象有两种成员，一种是属性，通过object.a 语法访问；另外一种是方法，通过object.b() 语法访问。（如果方法也是对象，它是什么样的对象呢？就是Function对象，那么Function对象为什么需要“()”的后缀呢？ 经过测试得, 将一个函数对象赋值给一个变量，“()”的含义是执行其内容，而没有”()”时，是直接打印的函数体本身，也就意味着() 可能是代表着是一个执行方法，当函数对象调用这个方法时，可能就是按照 apply(myObject,myArray) 类似的方式执行，第一个参数标志位置，第二个参数标志执行时的参数）

3.javascript是面向对象的语言，但是JavaScript不会创建类。JavaScript是基于prototype的

4.Number

4.1所有JavaScript的数字均为64位（52位值，11位指数，1为sign）

4.2整数不使用小数点或指数计数法时，最多15位，小数的最大位数是17，但是浮点运算并不是100%准确的 （ - -！ 这几个位是怎么得出来的来着~~~~~）

4.3JavaScript支持8进(0533)和16进制(0xFF)

var num = 16;

num.toString(16);

num.toString(8);

num.toString(2); //这里才是常规数字最快的转2进制的办法~~~~~~~~（比查啥子网站都快)

4.4Infinity也分正负,也是number类型,NaN也是number，可以用isNaN来判定。

4.5可以通过 new Number(123)来创建一个与从number初始化池子里的数字不一样的object

4.6几个处理时可能用到的非常规方法

toExponential() //指数计数

toFixed() //数字转字符串，可以指定小数点后的位数

toPrecision() //可以把数字格式化成指定长度，不足的补0，多余的填0

toString() //进制转换

valueOf() //返回number对象
