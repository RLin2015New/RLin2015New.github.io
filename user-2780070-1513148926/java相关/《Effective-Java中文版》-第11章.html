第11章 序列化

说序列化的意思是说java提供的对象序列化API，序列化核心解决的问题是将对象转化为字节流，通常之后的操作是进行传输或者持久化，再在需要用到的地方通过反向序列化来恢复对象，以达成这类“恢复对象场景”的需求。
